# 절대 규칙 — 반드시 지킬 것

## ⚠️ 코드 수정 전 필수 절차 (MUST)
- NEVER Edit/Write/Bash로 코드를 수정하기 전에 반드시 사용자에게 변경 계획을 단계별로 설명하고 명시적 승인을 받을 것
- 승인 없이 바로 툴을 호출하는 것은 절대 금지
- Read/Glob/Grep으로 코드를 파악하는 것은 승인 없이 가능

## 코드 규칙
- 관련 없는 코드는 절대 건드리지 말 것
- 실제 업계 표준을 준수할 것
- 수정한 코드에 기능 주석을 달 것
- 더 좋은 대안이 있으면 먼저 추천할 것
- 코드 수정 후 반드시 디버깅하고 오류 확인할 것
- 보안 이슈는 반드시 고칠 것

## CSS / UI 규칙
- 테마: https://bootswatch.com/vapor/ 참고
- Bootstrap 기본 클래스를 최대한 재사용할 것 (대체 불가할 때만 예외)
- 반응형 필수 — 모바일과 PC를 구분하여 작성할 것
