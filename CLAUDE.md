# 규칙

- 정할 때 관련없는 코드는 건드리지 말 것수
- 실제 업계 표준을 준수하며 표준을 꼭 지키도록 할 것
- 코드에 해당 기능이 무엇을 하는지 주석을 달 것
- 더좋은 대안이 있다면 추천할 것
- 코드 수정후엔 디버깅을 하고 오류를 꼭 확인할 것
- 수정전에 사용자에게 단계적으로 설명을 하고 검토를 받을 것
- 반응형으로 만들것 모바일과 pc 는 구분하여 작성할것
- css를 할떄 https://bootswatch.com/vapor/ 이사이트를 참고할것 테마임
- css는 최대한 부트스트랩기본 코드를 재사용할것 대체가 안될때는 제외
- 보안이슈는 꼭 고칠것.
